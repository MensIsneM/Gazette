## ✨ About

A simple, privacy-focused tool to generate documents required for **Central Gazette name and gender change (India)**.

- 🧠 Smart transition logic (MTF, FTM, MTT, FTT, TTM, TTF)
- 📄 Generates affidavit, newspaper ad, CD content
- ⚡ Works fully offline
- 🔒 No data is collected or stored
