I need your Support

This project was made by a trans person trying to survive the same system everyone using this tool is struggling through.

I built this because I was tired of seeing people get exploited by agents, lose money over formatting mistakes, or spend weeks trying to understand a process that barely has proper documentation online.

The site itself is hosted for free, but surviving is not.

Right now I am unemployed, financially unstable, and trying to continue transition-related medical care while dealing with housing insecurity. A lot of the time, development on this project happens while actively worrying about food, medication, or whether I’ll still have stable shelter in the near future.

Despite that, I still keep working on this because I know how terrifying and isolating this process can feel, especially for trans people with little support.

If this tool helped you:
* save money
* avoid agents
* understand the process
* generate documents
* or feel less alone navigating Gazette procedures

please consider supporting me.

Donations directly help me:
• afford medication and transition-related expenses
• continue maintaining and improving this tool
• work on new guides and utilities
• survive long enough to keep building resources like this for the community

Even small support genuinely helps more than you probably realize.

Thank you.
